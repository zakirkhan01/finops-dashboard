## 📌 Overview

The **FinOps Dashboard 2026** is a modern, single-file interactive web application designed to simulate **real-time cloud cost monitoring, anomaly detection, and unit economics tracking**.

This project demonstrates how engineering teams can move beyond traditional billing and adopt **continuous cost observability** in the age of AI and high-scale cloud systems.

---

## ⚡ Key Features

### 🔥 Cost Anomaly Heatmap

* 24x7 grid visualization of cost spikes
* Gradient-based severity (Green → Amber → Red)
* Tooltip insights (region, service, spike %)

### 📊 Unit Economics Live Chart

* Real-time simulation of:

  * Cost per Request
  * Revenue per Request
* Smooth animations with glowing effects
* Auto-updating every 2 seconds

### 🌐 Cross-AZ Traffic Visualizer

* Animated Availability Zones (AZ-1, AZ-2, AZ-3)
* Live traffic flow simulation
* Highlights expensive cross-AZ transfers

### 🧠 AI Advisory Engine

* Multi-factor analysis:

  * Cloud spend
  * Cost anomalies
  * Idle resources
* Dynamic insights like:

  * “Critical: Cross-AZ traffic is high”
  * “Warning: Idle GPU detected”

### ⚙️ Node Health Honeycomb Grid

* Interactive hex grid visualization
* Real-time status:

  * 🟢 Active (pulse animation)
  * 🟡 Warning
  * 🔴 Failed (blinking)
* Latency displayed per node

### 📡 Team Activity Stream

* Live scrolling activity feed
* Severity-based colors:

  * Green → Success
  * Amber → Warning
  * Red → Critical
* Real-time event injection

### 🧾 FinOps Checklist Panel

* Interactive checklist
* Tracks optimization completion %
* Includes:

  * Egress Audit
  * Unit Economics Baseline
  * Zombie Resource Cleanup
  * Spot Usage
  * Tag Compliance

## 💡 The $2M Cloud Leak Story

This dashboard is inspired by a real-world FinOps scenario where:

* Cross-AZ traffic caused massive hidden costs
* Cost per request jumped drastically
* No system errors — only billing explosion

### 🔍 Root Cause

Unoptimized **cross-availability zone data transfer**

### 🛠 Fix Applied

* AZ-aware routing
* Traffic localization
* Reduced cost + improved efficiency
